# ⚡ Rendr

**AI Product Demo Maker - Built for HeyGen Hackathon**

Rendr is an AI-powered platform that automatically transforms raw screen recordings into polished, high-converting product demonstrations using **HyperFrames** (AI Video Generation).

## 🚀 Features

- **HyperFrames Engine**: Understands your UI and generates cinematic camera movements, zooms, and highlights automatically.
- **Desktop First**: A native desktop application for high-performance screen recording with low overhead.
- **Extension (Coming Soon)**: One-click demo generation directly from your web browser.
- **AI Automation**: No more manual editing. Rendr identifies key moments in your recording and highlights them for you.
- **Premium Aesthetics**: Built with a focus on speed, clarity, and premium design.

## 🏆 HeyGen Hackathon

This project was built for the **HeyGen Hackathon - Build anything with AI video generation**. Rendr leverages cutting-edge AI video technology to solve the tedious process of manual video editing for product demos.

## 🏁 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## 🛠️ Tech Stack

- **Framework**: [Next.js 16+](https://nextjs.org)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Engine**: HyperFrames (AI Video Generation)
